## 📅 Day 1 — 3 Powerful JS One-Liners

### 🔁 Reverse a String

```js
const reverse = str => [...str].reverse().join('');
```

### 🧹 Remove Duplicates

```js
const unique = arr => [...new Set(arr)];
```

### 📦 Flatten Arrays

```js
const flat = arr => arr.flat(Infinity);
```
